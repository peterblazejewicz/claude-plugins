# Avalonia Project Structure Review

Run a comprehensive review of the current Avalonia or MAUI project structure.

## What This Command Does

1. **Analyzes** the project for hardcoded colors, fonts, and spacing values
2. **Identifies** design token extraction opportunities
3. **Evaluates** folder organization and separation of concerns
4. **Recommends** target structure based on project size
5. **Creates** a phased migration plan

## Usage

Run `/avalonia-review` in an Avalonia or MAUI project directory.

## Invoke Skill

Use the `avalonia-dev:review` skill to perform this analysis.
